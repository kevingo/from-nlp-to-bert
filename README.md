# From NLP to Bert

In this repositoy, I will make some notes on the introductory on NLP to Bert.

## Agenda

- Introduction to NLP
  - Tradition NLP (WordNet / one-hot encoding)
  - lexical analysis
  - Syntactic analysis
  - Semantic analysis
- Basic NLP
  - tf-idf
  - bag-of-word (BoW)
  - n-gram language model
  - limitation of BoW and n-gram language model
- Deep NLP
  - Word2Vec
  - RNN
  - Sequence-to-sequence
  - Attention Model
  - Transformer
- Bert
- Conclusion
